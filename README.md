# ⚡️ 3D 趣味电路实验室 (Circuit Fun 3D)

## 📖 项目简介
本项目是一个专为小学生设计的 **3D 交互式电路启蒙教学课件**。通过生动逼真的 3D 效果，让孩子们在浏览器中就能像玩积木一样搭建电路，安全、直观地学习基础物理知识。

我们的目标是：**让每一个孩子都能轻松点亮心中的“小灯泡”！** 💡

## ✨ 核心功能

### 1. 沉浸式 3D 实验台
- **逼真的组件模型**：包含干电池 🔋、小灯泡 💡、开关 🔘、导线 〰️ 等基础元件，具备 3D 视觉效果。
- **自由视角**：支持 360 度旋转、缩放观察电路连接细节。

### 2. 自由拖拽与搭建 (自定义电路)
- **拖拽式操作**：从工具箱中拖拽元件到实验台。
- **智能吸附连接**：导线靠近端口自动吸附，降低操作难度，模拟真实接线体验。
- **自定义布局**：支持孩子发挥想象力，搭建各种形状的电路图。

### 3. 实时物理仿真
- **点亮效果**：当电路闭合且电压足够时，灯泡会发出柔和的光芒（支持发光特效）。
- **短路警告**：如果发生短路，会有可爱的警告动画，教导安全用电知识。
- **状态反馈**：开关的闭合/断开有动画反馈。

### 4. 教学引导模式
- **循序渐进的任务**：从“点亮一个灯泡”开始，逐步解锁更有趣的挑战。
- **知识小百科**：点击组件可查看简单的科普介绍（语音+文字）。

## 📚 课程大纲 (教学路线图)

### 第一阶段：初识电的朋友 (入门)
- **第 1 课：认识神奇的元件**
    - 介绍电池（电源）、灯泡（用电器）、导线（传输）。
    - 3D 展示各部件，了解正负极。
- **第 2 课：点亮第一盏灯**
    - 任务：将电池和灯泡用导线连接起来。
    - 目标：理解“回路”的概念。

### 第二阶段：我是电路小指挥 (进阶)
- **第 3 课：开关的魔法**
    - 加入开关组件。
    - 任务：制作一个可以控制灯泡亮灭的电路。
- **第 4 课：电池加把劲**
    - 串联电池实验：两节电池会让灯泡更亮吗？

### 第三阶段：小小工程师 (挑战 - 待开发)
- **第 5 课：串联与并联**
    - 多个灯泡的连接方式。
    - 观察亮度变化，理解分压与分流（简化概念）。

## 🛠 技术栈规划
- **前端框架**: React
- **3D 引擎**: Three.js / React Three Fiber (R3F)
- **物理引擎/状态管理**: Zustand (用于管理电路连接状态)
- **UI 组件库**: Tailwind CSS / Ant Design (用于控制面板)

## 🚀 快速开始 (开发中)

```bash
# 克隆项目
git clone https://github.com/your-username/circuit-fun-3d.git

# 进入目录
cd circuit-fun-3d

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
